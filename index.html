What is tee.guru?

tee.guru is an ongoing bug bounty competition focused on Trusted Execution Environments (TEEs). Security researchers, developers, and enthusiasts come together to create, maintain, and submit bugs for TEE applications developed on modern TEE platforms.

TEEs provide isolated execution environments that protect code and data from the main operating system and other applications. Finding vulnerabilities in these secure enclaves is critical for advancing the security of hardware-based protection mechanisms.

TEE Platforms We Cover

Intel SGX AMD SEV ARM TrustZone Intel TDX RISC-V Keystone AWS Nitro Enclaves

How It Works

  • Create: Develop TEE applications that challenge security researchers
  • Maintain: Keep your TEE apps updated and running for the community
  • Submit: Find and report bugs in TEE applications to earn recognition
  • Collaborate: Work with fellow researchers to improve TEE security
